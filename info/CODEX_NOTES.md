# Codex Notes

## Проект

- Сайт для ателье в Орске
- Это сайт для мамы пользователя
- Сайт должен ясно показывать и пошив, и ремонт, и подгонку по фигуре
- Текст нужен простой, человеческий, без рекламной воды и без служебных объяснений

## Бизнес-данные

- Телефон: `+7 909 606-67-78`
- Адрес: `Проспект Ленина, 83, Орск`
- Часы работы: `будни с 11:00 до 18:00`
- Дополнительно: можно договориться на более раннее или позднее время по предварительному звонку
- Каналы связи: телефон, SMS, мессенджеры, VK

## Что важно в подаче

- Не уводить сайт только в тему ремонта: пошив тоже должен быть заметен
- Услуги нужно объяснять на бытовом русском языке
- Цены показывать как ориентир, без обещания точной суммы заранее
- Контакты и способ обращения должны читаться сразу

## Что пользователь уже забраковал

- Декоративные SVG с пуговицами, ножницами, нитками и псевдо-швейными линиями
- Нереалистичные украшения, которые выглядят как клипарт
- Случайные линии и декоративные хвосты под заголовками секций
- Слабую визуальную проверку без реального пролистывания страницы по экрану

## Текущие выводы по дизайну

- Hero уже собран заметно лучше, чем остальная страница
- Переработаны секции услуг, популярных запросов, цен, workflow и контактов
- Дополнительно усилены hero и блок примеров работ
- Повторяемость одинаковых белых карточек уменьшена, но сайт ещё можно сделать плотнее и увереннее
- Швейная тема сейчас лучше работает через строчки, лекальные сетки и спокойные рамки, а не через прямые иконки
- На мобильной версии ритм уже лучше, но при следующих правках всё равно нужно смотреть всю страницу до футера

## Как проверять дальше

- После правок HTML или CSS обязательно открывать страницу в браузере
- Смотреть страницу по экрану сверху вниз, а не только по DOM или full-page скрину
- Отдельно проверять десктоп и мобильную ширину
- Перед финалом не оставлять локальные изменения без коммита
