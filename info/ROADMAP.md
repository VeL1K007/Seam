# Roadmap

## Ближайший приоритет

Сделать сайт не просто чистым, а цельным и взрослым по виду, без кустарного декора.

## Что доработать дальше

1. Добить первый экран до более сильной и запоминающейся композиции
2. Подумать, нужен ли ещё один акцентный приём для блока с примерами работ
3. Проверить, не слишком ли затянут блок с примерами работ на мобильной версии
4. При следующем проходе выровнять плотность отступов между соседними секциями
5. Решить, нужен ли ещё один тихий фирменный мотив ателье без возврата к клипарту

## Что не делать

- не возвращать декоративные ножницы, пуговицы, нитки и подобный клипарт
- не добавлять служебные объяснения в пользовательский текст
- не плодить мелкие украшения без реальной пользы для композиции
- не считать задачу закрытой без визуальной проверки в браузере
