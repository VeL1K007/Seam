# Roadmap

## Ближайший приоритет

Сделать сайт не просто чистым, а цельным и взрослым по виду, без кустарного декора.

## Что доработать дальше

1. При следующем проходе выровнять плотность отступов между соседними секциями
2. Решить, нужен ли ещё один тихий фирменный мотив ателье без возврата к клипарту
3. При желании отдельно пересобрать верхнее мобильное меню и первый экран ещё дороже по подаче
4. Посмотреть, не нужен ли более сильный финальный блок контактов или CTA перед футером

## Что не делать

- не возвращать декоративные ножницы, пуговицы, нитки и подобный клипарт
- не добавлять служебные объяснения в пользовательский текст
- не плодить мелкие украшения без реальной пользы для композиции
- не считать задачу закрытой без визуальной проверки в браузере
