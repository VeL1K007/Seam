# Wishes

## По визуальному стилю

- сайт должен выглядеть аккуратно, взросло и собранно
- нельзя решать тему ателье дешёвым клипартом
- любые декоративные мотивы должны быть правдоподобными и очень дозированными
- если украшение спорное или выглядит странно, его лучше убрать, чем оставлять

## По процессу

- после CSS и HTML правок страницу нужно реально открывать и пролистывать
- нужно смотреть не только первый экран, но и всю страницу до футера
- обязательно проверять мобильную версию отдельно

## По контенту

- на сайте не должно быть служебных комментариев и текстов для разработчика
- текст должен быть обычным, понятным клиенту
- контакты, часы и способ обращения должны быть заметными
- пошив не должен теряться на фоне ремонта
