# Info

Эта папка нужна, чтобы хранить рабочий контекст по сайту и не терять договорённости между доработками.

Что лежит внутри:

- `CODEX_NOTES.md` — живая выжимка по проекту, текущему состоянию и важным замечаниям
- `ROADMAP.md` — ближайшие доработки по дизайну, структуре и контенту

Правила обновления:

- после заметных изменений по дизайну или структуре обновлять заметки
- фиксировать не только сделанное, но и то, что пользователь явно забраковал
- не записывать абстракции, писать конкретно про этот сайт и этот проект
